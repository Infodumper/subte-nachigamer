# 🚇 Subte Nachi - Web App

¡Hola! Esta es la versión "ordenada" y mejorada de la app que empezó Stitch para Nachi. 

## 🚀 Cómo ver la App
1. Solo tenés que abrir el archivo **`index.html`** en cualquier navegador (Chrome, Safari, etc.).
2. ¡Listo! Ya podés navegar entre las secciones usando la barra de abajo.

## 📂 Estructura del Proyecto
- **`index.html`**: El corazón de la app.
- **`js/app.js`**: El cerebro que maneja el cambio de pantallas.
- **`js/views.js`**: Contiene todo el diseño de las pantallas (aquí puse lo que hizo Stitch, pero más lindo y grande para Nachi).
- **`assets/screenshots`**: Guardé las capturas originales de Stitch por si las necesitás.

## ✨ Mejoras para Nachi
- **Botones Gigantes**: Todo es fácil de tocar.
- **Colores Claros**: Las líneas tienen sus colores reales y brillantes.
- **Mensajes Amigables**: En la sección "Nachi" hay mensajes directos para él.
- **Estado Visual**: Iconos de check ✅ y cruces ❌ muy claros.
